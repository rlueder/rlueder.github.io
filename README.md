# rlueder.github.io

## Overview

This repo is an attempt to acquaint myself with different programming languages and respective web frameworks.

The list of supported languages/frameworks currently includes:

- React (JS)
- Django (Python)
- Hugo (Go)
- Phoenix (Elixir)
- Yew (Rust)

Upcoming languages/frameworks will include:

- Javalin (Kotlin)
- Reagent (Clojure)

## Installing

### React (JavaScript)

`$ cd js_react`

`npm install`

`npm start`

Sever will run at `http://localhost:3000/`

#### Documentation

- https://create-react-app.dev/docs/getting-started
- https://github.com/facebook/create-react-app

### Django (Python)

`$ cd py_django`

`$ python manage.py runserver`

Server will run at `http://localhost:8000/`

#### Documentation

- https://docs.djangoproject.com/
- https://github.com/django/django

### Hugo (Go)

`$ cd go_hugo`

`$ hugo server -D`

Server will run at `http://localhost:1313/`

#### Documentation

- https://gohugo.io/
- https://github.com/gohugoio/hugo

### Phoenix (Elixir)

`cd ex_phoenix`

`$ mix phx.server`

Server will run at `http://localhost:4000/`

#### Documentation

- https://hexdocs.pm/phoenix/
- https://github.com/phoenixframework/phoenix
